# Financial Project Planning & Forecasting Suite

This repository contains an end-to-end data science solution designed to analyze historical financial variance and generate high-accuracy sales forecasts. The project integrates a Machine Learning pipeline with an interactive web dashboard to bridge the gap between projected budgets and actual market performance.

## 📌 Project Overview
The objective of this project is to solve common financial planning inefficiencies—such as overstocking or missed revenue surges—by providing two core tools:
1.  **Historical Audit (Power BI):** A diagnostic tool that identifies specific regions or categories where past financial guesses were inaccurate.
2.  **Predictive Engine (Streamlit + Prophet):** A machine-learning-powered application that allows users to run "What-If" simulations and view 12-month future projections using the powerBI dashboard.

## 🛠️ Tech Stack
- **Data Processing:** Python, Pandas, Numpy
- **Machine Learning:** Meta Prophet (Time-Series Forecasting)
- **Web Interface:** Streamlit
- **Visualization:** Plotly (Interactive Charts), Power BI (Historical Audit)
- **Environment:** VS Code, Jupyter Notebooks
